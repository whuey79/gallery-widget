Javascript image gallery widget

Created 8/21/14 by Will Huey - wdhuey@gmail.com

The top widget uses thumbnails to control what's displayed on the main viewer. The default image for the main viewer is the first image in the filmstrip.
The bottom widget changes images when the image is clicked. Images rotate back to the beginning after the length is met.
The third widget shows all 6 images in thumbnail mode.
